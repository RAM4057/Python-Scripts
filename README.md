# Python Scripts

CSV report generator and OCR text extractor for macOS.

## CSV Report Generator

1. Place CSV file in the same folder as the script
2. Run: `python3 csv_report.py your_file.csv`

## OCR Text Extractor

1. Install Tesseract: `brew install tesseract`
2. Install dependencies: `pip install -r requirements.txt`
3. Place image file in the same folder as the script
4. Run: `python3 ocr_extract.py your_image.jpg`
5. Extracted text will be saved as `your_image_extracted.txt`

Done!
